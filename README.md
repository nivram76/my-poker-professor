# PokerSense

PokerSense is a guided poker hand review app with an AI coach. The MVP starts
with a UI-first, mock-data workflow before adding the FastAPI, PostgreSQL, and
Redis backend.

## Current Scope

- Next.js App Router
- TypeScript
- Tailwind CSS
- Railbird-inspired three-panel analysis workspace at `/`
- Structured hand builder at `/create`
- Stubbed voice input
- Mock sessions, hands, optimal plays, and coach cards

## Local Development

```bash
npm run dev
```

Open [http://localhost:3000/create](http://localhost:3000/create).

## Checks
